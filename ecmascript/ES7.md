### **ES7 (ES2016)**

1. **`Array.prototype.includes`**  
   Check if an array contains a value.

   ```javascript
   const arr = [1, 2, 3];
   console.log(arr.includes(2)); // true
   ```

2. **Exponentiation Operator (`**`)\*\*  
   Raise a number to a power.
   ```javascript
   console.log(2 ** 3); // 8
   ```
